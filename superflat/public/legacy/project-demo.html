GAME DESIGN

King's Command

A turn-based tactics game where players command autonomous AI soldiers through natural language orders. Exploring the boundary between strategic intent and emergent behaviour.

// Hero Code — spans full viewport
const orders = await parseNaturalLanguage(input);

  const intent = extractIntent(order);
  execute(intent);

The premise

What if you could command units in a tactics game the way you'd brief a team in real life? "Flank from the left, hold fire until they're in range, retreat if outnumbered." The game became a vehicle for exploring how LLMs interpret tactical language.

Strategy emerges from the gap between what you say and what the system understands.

Process & artefacts

Early iterations had the LLM generating moves directly. We switched to intent extraction + deterministic execution so players could debug "why did it do that?"
